# Scala language custom image
Dev environment for awesome https://www.scala-lang.org/

## Details
Installed packages:
* SDKMAN 5.11,
* AdoptOpenJDK 11,
* Scala 2.13.5

It is based on debian:buster-slim.

## How to use
```bash
docker container run -it --rm kraftdorian/scala bash
```
